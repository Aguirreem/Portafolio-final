Práctica 4: Brazo Robot

Alberto Aguirre Medina
Ing. Mectrónica
Gustavo Isaac Becerril Villamil
Ing. Mecatrónica
Fecha:Enero 31, 2025
Arduino Electrónica Robótica

- Resumen -

En este proyecto, se diseñó y construyó un brazo robótico utilizando MDF como material estructural, controlado mediante Arduino y servomotores. Se programó su funcionamiento para ejecutar movimientos precisos a través de señales enviadas desde un microcontrolador, con control manual mediante potenciómetros y automatizado mediante programación. Este brazo robótico representa una introducción al uso de la mecatrónica en sistemas de automatización y robótica.

- Introducción -

El avance de la robótica ha permitido el desarrollo de brazos robóticos utilizados en diversas aplicaciones industriales y educativas. Este proyecto busca aplicar principios de electrónica, mecánica y programación para diseñar un brazo robótico funcional utilizando MDF como estructura base. Se utilizarán servomotores para controlar el movimiento y un Arduino como unidad central de procesamiento.

- Materiales -

  • Placa Arduino Uno
  • MDF de 3mm o 5mm para la estructura
  • Servomotores SG90 o MG995
  • Tornillería y tuercas
  • Potenciómetros
  • Cables de conexión, jumpers
  • Protoboard
  • Fuente de alimentación externa de 5V-6V para los servomotores
  • Computadora con Arduino IDE instalado

Simulación

Antes de construir físicamente el brazo, se realizó una simulación en software como Tinkercad o Fusion 360 para validar el diseño mecánico y el código de control de los servomotores.

Diseño

Se diseñó la estructura del brazo en un software CAD como AutoCAD o Fusion 360, considerando articulaciones móviles y soportes para servomotores. Luego, las piezas fueron cortadas en MDF con láser

Construcción

Las piezas de MDF fueron ensambladas utilizando tornillos, asegurando estabilidad y libertad de movimiento en las articulaciones. Los servomotores fueron montados en las uniones clave para permitir la movilidad.

- Resultados -

Se logró construir un brazo robótico funcional con MDF, capaz de moverse en diferentes direcciones mediante servomotores. La programación permitió controlar el brazo con potenciómetros y ejecutar secuencias de movimiento automatizadas. El uso de una fuente de alimentación externa mejoró el rendimiento de los servomotores, evitando sobrecargas en la placa Arduino. Este proyecto brinda una base sólida para el desarrollo de sistemas robóticos más avanzados, permitiendo la exploración de temas como la visión artificial, la inteligencia artificial y la automatización industrial.

- Conclusiones -

La construcción de un brazo robótico con MDF, Arduino y servomotores permitió integrar conocimientos de mecánica, electrónica y programación en un solo proyecto. A lo largo del desarrollo, se comprendió la importancia del diseño estructural para garantizar estabilidad y movilidad, así como la necesidad de una correcta gestión de la energía para un funcionamiento óptimo de los servomotores.
