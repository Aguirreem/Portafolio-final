Práctica 1: Arduino Servomotores

Alberto Aguirre Medina
Ing. Mectrónica
Gustavo Isaac Becerril Villamil
Ing. Mecatrónica
Fecha:Enero 31, 2025
Arduino Electrónica

- Resumen -

En esta actividad de Arduino Básico, se trabajó con servomotores para comprender su funcionamiento y control mediante Arduino. Se realizaron diversas prácticas, desde el control básico de un servomotor hasta la manipulación simultánea de dos servos con potenciómetros. Además, se implementó una fuente de alimentación externa para los circuitos realizados. Estas actividades permitieron desarrollar habilidades en el manejo de motores, lo cual tiene aplicaciones en la ingeniería mecatrónica, como los brazos robóticos.

- Introducción -

- Materiales -

  • Placa Arduino Uno
  • LEDs
  • Resistencias de 220Ω y 1kΩ
  • Cables de conexión, jumpers
  • Protoboard
  • Servomotores
  • Computadora con Arduino IDE instalado
  • Fuente de alimentación

- Desarrollo -

Se realizaron diversas actividades para comprender el control de servomotores con Arduino. Se comenzó con el movimiento básico de un servomotor y luego se programó para moverse entre distintas posiciones predeterminadas. Posteriormente, se exploró el control de un servomotor mediante la lectura de un potenciómetro, seguido de la implementación de un sistema en el que un solo potenciómetro controlaba dos servomotores. También se desarrolló una configuración en la que dos servomotores eran controlados de manera independiente por dos potenciómetros. Finalmente, se experimentó con el uso de una fuente externa de alimentación para garantizar un funcionamiento adecuado de los servomotores. Cada ejercicio implicó la correcta conexión de los componentes y la programación en el lenguaje de Arduino para lograr el control preciso del movimiento de los servomotores.

Simulación

Se utilizó el simulador Tinkercad para probar los circuitos y asegurarse de su correcto funcionamiento antes de implementarlos en la parte física. Esto permitió comprender la programación del circuito y por ende mejorar la comprensión de la relación entre el código y el comportamiento del servomotor.

Diseño

Los circuitos utilizando servomotores y potenciómetros se diseñaron siguiendo ejemplos de referencia de Tinkercad, asegurando una organización adecuada de los componentes en el protoboard y comprensión del circuito eléctrico. De igual manera se tomó en cuenta el voltaje y la corriente de la fuente alimentación, manipulándola adecuadamente para hacer funcionar los servomotores y para evitar quemar componentes o la placa Arduino.

Construcción

Se ensamblaron los circuitos físicamente, conectando los servomotores y potenciómetros a la placa Arduino. Se cargaron los programas desde el Arduino IDE y se logró el correcto funcionamiento de cada circuito.

- Resultados -

Se alcanzó un control eficiente de servomotores con Arduino. Se observó la influencia de los potenciómetros en la posición de los servos y la importancia de utilizar una fuente de alimentación externa para evitar inestabilidad en el sistema. La actividad permitió comprender cómo integrar estos dispositivos en proyectos de robótica y automatización

- Conclusiones -

El manejo de servomotores controlados con Arduino tiene mucha presencia en la ingeniería mecatrónica, ya que permite el desarrollo de sistemas automatizados con control de movimiento. A partir de esta actividad, se cuenta con la base necesaria para construir proyectos más avanzados. Por ejemplo, pueden aplicarse estos conocimientos para diseñar y construir un brazo robótico, integrando servomotores, potenciómetros y una fuente de alimentación externa, logrando un sistema que pueda manipular objetos de manera autónoma o controlada manualmente.
