Practica 5: Creacion pagina web

- Resumen -

Este proyecto tiene como objetivo crear una página web básica utilizando HTML y alojarla en GitHub a través de su servicio GitHub Pages. Se aprenderá a desarrollar una estructura básica de una página web, configurar un repositorio en GitHub y desplegar la página en línea.

- Introducción -

La creación de una página web básica en HTML es una habilidad esencial para desarrolladores y diseñadores web. GitHub Pages permite alojar proyectos web de manera gratuita y sencilla, usando archivos HTML, CSS y JavaScript. En este ejercicio, se desarrollará una página web estática básica y se alojará en GitHub para que sea accesible desde cualquier navegador web.

- Desarrollo -

Para crear y alojar una página web básica en HTML en GitHub, primero debes crear un repositorio en GitHub y subir un archivo index.html que contenga el código HTML de tu página. En este archivo, puedes incluir una estructura básica como el título de la página y algún contenido en el cuerpo. Una vez que hayas subido el archivo al repositorio, habilita GitHub Pages en la sección de configuraciones del repositorio, seleccionando la rama main como fuente. Esto generará una URL pública donde podrás acceder a tu página web desde cualquier navegador. Finalmente, verifica que la página se visualice correctamente accediendo a la URL proporcionada por GitHub Pages.

Diseño

Antes de cargar la página en GitHub, puedes simularla en tu máquina local para asegurarte de que todo funcione correctamente. Para ello, abre el archivo index.html en tu navegador web. Revisa que el contenido y el diseño sean los esperados.

Construcción

La construcción de la página web consiste en escribir el código HTML y luego subirlo al repositorio en GitHub. Se debe asegurar que el archivo index.html esté correctamente nombrado y ubicado en la raíz del repositorio para que sea reconocido por GitHub Pages.

- Resultados -

Después de habilitar GitHub Pages, podrás acceder a tu página web en línea mediante una URL generada por GitHub. Esta página será accesible desde cualquier navegador y se verá como la estructura básica que creaste. Puedes compartir el enlace con otras personas o continuar desarrollando tu página web con más contenido y estilos.

- Conclusiones -

Crear una página web básica en HTML y alojarla en GitHub Pages es un proceso sencillo y gratuito. GitHub Pages es una excelente opción para proyectos personales, portfolios o para compartir contenido estático en línea. Este ejercicio proporciona una introducción práctica al desarrollo web y al uso de herramientas como GitHub para desplegar proyectos.
